Welcome to the guide for modding Lethal Company for GHC Gaming Club's Friday Fright Night event!

To get started, you will need to install the mod manager. The application itself is called r2modman and will be a UI for managing all mods from the Thunderstore.
Feel free to keep it and get familiar with it, as it supports several games more than just Lethal Company!

Installing r2modman

  1. Open this link: https://thunderstore.io/c/lethal-company/p/ebkr/r2modman/
  2. Click the Manual Download button.



  3. A ZIP file named ebkr-r2modman-3.1.50 will install. Open and extract it to its own folder.




  4. Once extracted, open the new folder and run the r2modman-Setup-3.1.50 application file.



    • It is possible you may get a popup on Windows at this stage -- titled Windows protected your PC.
      This is only because of publisher IDs missing from the file data. It is safe to install. Simply click More info and then Run anyway.



    • If you want to install this for all users on your PC, make sure to right click and open the program as an administrator to have the option during installation.

  5. The r2modman Setup Wizard will launch to Choose Installation Options. Choose your preference and click Next.



  6. Now you will have the opportunity to change the destination of the install. It will default to C:\Users\USERNAME\AppData\Local\Programs\r2modman. When done, click Install.



  7. Once your installation is complete, leave the Run r2modman box checked and hit Finish to close setup.



  8. Welcome to r2modman! üéâ

Modding Lethal Company

Now that you have r2modman, you'll need to use it to get your mods. The easiest part about this is that we've already made a modpack for you!

Here's the steps you need to take to install the modpack:

  1. First you need to select your game. Scroll or search for Lethal Company and then click Select game.
    • Note: You can favorite the game to pin it to the top of the selection screen.



  2. Once you finish loading you will be on the profile selection page. Since we are installing a premade profile (AKA the modpack) you do not need to create a profile. Instead, select Import/Update.



  3. Select Import new profile.



  4. Select From code.



  5. Enter the profile code 01927e33-b197-b794-e4be-7b0a82fdd73a. Click on Continue.




  6. The list of mods to be installed will be displayed. Select Import.



  7. The name should default to "GC Friday Fright Night"; Select Create to download the mods. Do NOT close r2modman during this process.



  8. Once the install is finished, GC Friday Fright Night will be highlighted blue. If it's not, click on it. Then click Select profile.



From here, you can manage any mods on this profile. You will already have the required mods in our modpack enabled.



Tips and help

  • If there are any client side mods you wish to turn off, you can do so by toggling the blue switch on the right of the mod.
  • For extended configuration outside of the game engine, you can go to the Config editor to manage data settings.
